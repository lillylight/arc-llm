3D Visualization

To view the 3D visualization, please download the visualization file and open it with the Rerun viewer: 

rerun {{ layout_id }}.rrd
Instructions:
  1. Download the visualization file using the button above
  2. Install Rerun viewer if you haven't already: pip install rerun-viewer
  3. Open a terminal and navigate to the directory where you downloaded the file
  4. Run the command: rerun {{ layout_id }}.rrd

Note: Due to browser limitations, we cannot embed the 3D visualization directly in this page. You need to use the Rerun viewer to see the 3D visualization.
